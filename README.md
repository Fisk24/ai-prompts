# ai-prompts
a set of prompts designed to make LLM chatbots accomplish specific tasks

## How to use:
locate the prompt whoes name matches your use case. 
ie. "homework checker prompt"
and then use plain english to customize the prompt to match your specific requirements.

NOTE: some prompts assume that files will be uploaded along side them
For example: homework checker assumes that you will upload the assignment document 
in addition to one or more documents which contain your answers to the assignment.
